> ### If-else or if-else if-else

```js

        // multiple conditions with bool operators

        console.log(""); // <- print statement
    } else if (condition4) {
        /* 
            Can evaluate more conditions
        */
    } else {
        // execute anything that doesn't match if's
    }

```
